# Maintenance Cost
Both Cities and armies will have maintenance cost.
The end user will never directly receive information about the maintenence cost,
but we will calculate the maintenance cost based on the buildings (and also increases when buildings are upgraded).

Each building will have the following base maintenance cost:

* Barracks
    * Rations: 20/h

* Space Dock
    * Rations: 10/h
    * Oil: 25/h
    * Uranium: 1/h

* Nexus
    * Rations: 20/h

* Farmpod
    * Solarium: 10/h

* Cloning Chamber
   * Rations: 40/h

* Extractor
   * Coal: 15/h

* Reactor
   * Rations: 5/h

* Oil Pump
   * Rations: 5/h
 
* Parlement
   * Rations: 30/h

* Solar Generator
   * Rations: 2/h

* Material Lab
   * Rations: 5/h

* Plant Growth Accelerator
    * Rations: 20/h

* Electric Mine
    * Rations: 5/h
 
* Solar Lab
    * Rations: 5/h

For troops:
* Soldier Unit
    * Rations: 25/h
  
* Tank Unit
    * Rations: 40/h

* Assassin Unit
    * Rations: 20/h

* Medic Unit
    * Rations: 10/h
  
* Fighter Unit
    * Rations: 3/h
    * Oil: 10/h
  
* Bomber Unit
    * Rations: 3/h
    * Oil: 10/h
  
* MotherShip Unit
    * Rations: 60/h
